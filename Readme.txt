1. OVERVIEW
-----------
El objetivo era Desarrollar una aplicación móvil sencilla utilizando React Native que permita a
los residentes de un condominio reservar instalaciones compartidas.

2. KEY FEATURES
---------------
* Diseño de la interfaz de usuario: Diseñar la interfaz de usuario de la
aplicación. La aplicación debe incluir las siguiente pantalla:
	Una pantalla de inicio que muestra las instalaciones disponibles para reservar

* Desarrollo de la aplicación: Implementar la funcionalidad básica de la 
 aplicación utilizando React Native. La aplicación debe permitir a los usuarios
realizar la siguiente acción:
	Ver las instalaciones disponibles para reservar.

* Consideraciones adicionales que se tomaron para la creacion del proyecto
1.Para este caso práctico, no es necesario implementar un back-end real.
Puedes simular la interacción con un back-end utilizando datos estáticos.
2.Enfócate en la calidad del código, la escalabilidad y las mejores prácticas de
desarrollo.
3.Presta atención al diseño y la experiencia del usuario. La aplicación debe ser
fácil de usar y visualmente atractiva.


3. HOW TO INSTALL
-----------------
Como el objetivo era crear una 'pantalla de inicio' que muestra las instalaciones disponibles para reservar, alcanza con un 'git clone'.



5. TIPS
-------
* Para probar la escalabilidad del codigo puede modificar el archivo de data aumentando o quitando ambientes

* El proyecto tiene una estructura simple esta dividido en paginas y componentes (aunque solo fuera 1 se hizo para estructurar).
